Repositório de **estudos** para criação de **relatórios Excel (.xlsx) personalizados**, usando **pandas + openpyxl**: múltiplas abas, cabeçalho “bonito”, cores, congelamento de painel e ajustes visuais para ficar pronto pra compartilhar.

---

## 🎯 Objetivo

- Exportar DataFrames para Excel
- Criar **1 aba por relatório**
- Aplicar **cabeçalho do relatório** (título, data/hora, autor, etc.)
- Estilizar tabela (cores, fonte, bordas, freeze panes)

---

## 📁 Estrutura do repositório

```text
relatorio-xlsx/
├─ notebooks/
│  └─ bases.ipynb        # notebook principal de estudo/execução
├─ output/               # arquivos gerados (xlsx)
├─ .gitattributes
├─ .gitignore
└─ README.md
```
